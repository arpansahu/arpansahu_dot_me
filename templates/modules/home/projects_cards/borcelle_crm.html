Customer Relationship Manager Django app with Email Contact and sheduling facility.
  • Developed a Django-based Customer Relationship Manager (CRM) with email communication capabilities.
  • Implemented a progress bar for users to track task completion.
  • Admins receive notifications when scheduled messages are successfully delivered and can schedule broadcast notifications to all customers.
  • Deployed on Ubuntu VPS, containerized with Docker/Kubernetes. Enabled CI/CD pipelines for automated testing, building, and deployment.
  • Integrated Automatic Readme Generator Github
  • Deployed on Hostinger VPS
